Types of Markup Languages

There are many types of markup languages that phase in and out of relevancy. The most widely used are HTML, XML, and XHTML.

HTML - Hypertext Markup Language

HTML is a language used in the creation of websites. HTML uses a system of tags that help identify various components of the documents such as headings, paragraphs, and lists. HTML is a descendant of SGML which is a meta-language for building vocabularies. The current iteration of HTML is HTML5.

XML - eXtensible Markup Language

XML uses a similar vocabulary to HTML. Both use a system of markups to organize their content. The main difference between HTML and XML is that XML allows the user to create their own vocabularies with specific tags and rules.

XHTML - eXtensible Hypertext Markup language

XHTML was created to integrate HTML with XML. The purpose was to aid users in transitioning from HTML to XML. XHTML must conform to a XML syntax rather than HTML's syntax. XHTML has the benefits of the XML markup with the backwards compatibility of HTML.
