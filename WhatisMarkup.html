What is A Markup Language

A markup language is any coding language that uses tags to surround parts of the document you wish to enhance. Most markups close tags by using a forward slash indicating the end of the markup.

< p > This is an example of a basic html markup code < /p >.

As the example above shows, the < p > indicates the start of the syntax and the < /p > shows the end. This indicates to the web page what actions it needs to take. Tags need to be layed out in the proper order or the webpage will not know how to read the document.
Tags can indicate a variety of things to the browser. Some examples are:

  • Indicating paragraphs
  • Linking to external sites or pages within your document
  • Creating bold or italicized text
  • Creating forms

Contrary to popular belief, markup languages are not are not programing languages such as Python, JavaScript, or C#.
